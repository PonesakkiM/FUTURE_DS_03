College Event Feedback Analysis
This project analyzes student feedback collected from a college fest to uncover satisfaction trends, identify improvement areas, and understand students’ overall experience.
Project Overview

This project focuses on:

* Data Cleaning

* Exploratory Data Analysis (EDA)

* Sentiment Analysis

* Data Visualization (Charts & Radar Plot)

* Insights & Recommendations

 Dataset Information
The dataset contains 100 rows of student feedback with the following features:
Student_ID

Department

Rating

Food_Rating

Organization_Rating

Overall_Satisfaction

Event_Name

Event_Type

Year

Role

Role_Details

Feedback

Suggestions

 Tools & Libraries Used

Python

Google Colab

Pandas – Data Cleaning & Manipulation

Matplotlib & Seaborn – Visualizations

TextBlob / VADER – Sentiment Analysis

WordCloud – Text Visualization

NumPy
Steps Performed
1. Data Cleaning

Removed missing values

Converted data types

Removed duplicates

Cleaned & standardized text columns

2. Exploratory Data Analysis (EDA)

Rating distributions

Department-wise performance

Event-type satisfaction

Role-wise comparisons

3. Sentiment Analysis

Performed using TextBlob, classifying feedback into:

Positive

Neutral

Negative

4️. Visualizations

Included:

Bar Charts

Countplots

Boxplots

WordCloud

Radar Chart for multi-metric comparison

 Key Insights (Sample Findings)

Cultural events showed the highest satisfaction levels

Organizers gave higher organization ratings

Food quality received the most negative comments

Most feedback was overall positive
 Recommendations

Based on insights:

Improve food variety and quality

Add more seating/water facilities

Introduce more interactive events

Provide better communication and instructions to volunteers
